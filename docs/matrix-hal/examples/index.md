## MATRIX HAL Examples

### Download Examples

The following commands download and compile all the examples.

```bash
sudo apt-get install cmake g++ git
```

```bash
cd ~/
git clone https://github.com/matrix-io/matrix-hal-examples.git
cd matrix-hal-examples
mkdir build
cd build
cmake ..
make -j4
```
